# A-Share Analysis Skill (Markdown 托管模式)

本技能已重构为基于 **Markdown 行为准则** 的全流程量化分析工具。它作为股票市场的业务入口，引导用户进行深度的财务透视与舆情评估。

## 核心特性

### 1. 强制阶段化分析 (Stop & Report Workflow)
为了确保分析的透明度与准确性，Agent 必须严格按照以下阶段顺序执行，并在**每个阶段完成后立即向用户反馈详细结果**：
- **阶段 1：黑名单筛选**：剔除激进资本结构（长期融资净值不足）或现金流断裂的风险标的。
- **阶段 2：白名单发现**：基于 ROE (>10%)、BIT ROA、经营收益占比及现金含量锁定优质资产。
- **阶段 3：定投组合构建**：利用 **DCF 估值模型** 计算折扣率，并自动加入银行、非银、地产行业的强制品种（如招行、平安、万科等）。
- **阶段 4：舆情深度打分**：调用内置工具抓取实时资讯，利用金融分析师逻辑进行 -1 到 1 的量化评分。

### 2. 多源数据集成 (Multi-Source Execution)
- **财务与行情**：直接调用 **stock** MCP (`mysql`) 获取最新报表与 K 线数据。
- **实时资讯**：通过运行 `references/` 目录下的 Python 脚本抓取东方财富股吧的最新新闻与公告。
- **冲突处理**：内置了对矛盾舆情因素（如融资买入 vs 股东减持）的权重对比逻辑。

### 3. 市场引导与拦截
- **已上线**：🇨🇳 中国 A 股。
- **开发中**：🇺🇸 美股、🇭🇰 港股。
- **安全机制**：Agent 会自动识别市场，非 A 股需求将被直接拦截以保护 MCP 查询资源。

## 如何使用
您只需描述您的分析需求，Agent 会启动链式准则。例如：
- “帮我筛选出当前的定投组合，并根据最新舆情给它们打分。”
- “分析一下 600519 的财务健康度，它符合白名单标准吗？”
- “ROE > 15% 且有估值折扣的股票有哪些？”
